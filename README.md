# Projeto HTML + CSS

Este repositório contém uma página estática desenvolvida com HTML e CSS, com foco em layout simples de portfólio pessoal e navegação por abas sem JavaScript.

## Visão Geral

A interface combina:
- Uma barra superior de contatos com área de navegação (`Home`) e botões para perfis externos.
- Uma seção principal com conteúdo em abas, controladas por `input type="radio"` e `label`, exibindo textos informativos sobre Lorem Ipsum.

## Tecnologias Utilizadas

- HTML5
- CSS3

## Estrutura do Projeto

- `index.html`: marcação principal da página, incluindo barra de contatos, botões e conteúdo em abas.
- `style.css`: estilos visuais, layout, cores, tipografia e lógica de exibição das abas via seletores CSS.

## Como Executar

1. Abra o arquivo `index.html` diretamente no navegador.

Não há dependências externas, build ou instalação de pacotes.

## Funcionalidades da Página

- Layout centralizado da área principal com destaque visual por cores e sombras.
- Sistema de abas com três tópicos de conteúdo:
  - O que é Lorem Ipsum?
  - Por que nós o usamos?
  - De onde ele vem?
- Exibição condicional de conteúdo usando apenas CSS:
  - painéis ocultos por padrão;
  - painel ativo exibido conforme a aba selecionada.
- Botões de acesso para LinkedIn e GitHub no topo da página.

## Estilo e Organização Visual

- Uso de blocos com fundo colorido para separar navegação e conteúdo.
- Botões com cantos arredondados e contraste de cores.
- Abas com estado ativo visualmente destacado.
- Área de conteúdo com largura máxima definida para melhor leitura.
